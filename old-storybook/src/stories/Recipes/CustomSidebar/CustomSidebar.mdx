# Custom Sidebar

Your app may want to customize the sidebar to change the look and feel. Here are a few examples to help you get started

## Domain Registrar Example

```css
// Domains Sidebar Example
@mixin sidebar-domains-rail-global {
  .sidebar__brand {
    padding: 0 var(--_op-sidebar-spacing);
  }
}

@mixin sidebar-domains-drawer-global {
  .sidebar__content {
    padding: 0;
  }

  .sidebar__item {
    border-top-left-radius: 0;
    border-bottom-left-radius: 0;
  }

  .sidebar__section-divider {
    margin: var(--op-space-small) 0;
  }
}

.sidebar-domains {
  @extend %sidebar-global;

  // Normal
  --_op-sidebar-background-color: var(--op-color-background);
  --_op-sidebar-text-color: var(--op-color-on-background);

  // Active
  --_op-sidebar-icon-background-active-color: var(--op-color-primary-plus-seven);
  --_op-sidebar-icon-text-active-color: var(--op-color-primary-on-plus-seven);

  // Hover
  --_op-sidebar-icon-background-hover-color: var(--op-color-neutral-plus-seven);
  --_op-sidebar-icon-text-hover-color: var(--op-color-neutral-on-plus-seven);

  .sidebar__brand {
    padding: 0 var(--op-space-4x-large);
  }

  // Modifiers
  &.sidebar--drawer:not(.sidebar--responsive) {
    @include sidebar-domains-drawer-global;
  }

  &.sidebar--drawer.sidebar--responsive {
    @media (width > $breakpoint-medium) {
      @include sidebar-domains-drawer-global;
    }

    @media (width <= $breakpoint-medium) {
      @include sidebar-domains-rail-global;
    }
  }

  &.sidebar--rail {
    @include sidebar-domains-rail-global;
  }
}
```

## Performance Management Software Example

```css
// 16Six Sidebar Example
@mixin sidebar-16six-rail-global {
  .sidebar__brand {
    svg {
      width: 100%;
      padding: 0 var(--op-space-small);
    }

    .sidebar__brand-label {
      display: none;
    }
  }
}

@mixin sidebar-16six-drawer-global {
  .sidebar__item {
    border-radius: var(--op-radius-large);
    font-size: var(--op-font-small);
  }
}

.sidebar-16six {
  @extend %sidebar-global;

  --_op-sidebar-spacing: var(--op-space-x-small);

  // Normal
  --_op-sidebar-background-color: var(--op-color-primary-minus-five);
  --_op-sidebar-text-color: var(--op-color-primary-on-minus-five);

  // Active
  --_op-sidebar-icon-background-active-color: var(--op-color-primary-minus-three);
  --_op-sidebar-icon-text-active-color: var(--op-color-primary-on-minus-three);

  // Hover
  --_op-sidebar-icon-background-hover-color: var(--op-color-primary-minus-three);
  --_op-sidebar-icon-text-hover-color: var(--op-color-primary-on-minus-three);

  gap: var(--op-space-x-large);

  .sidebar__brand {
    display: flex;
    gap: var(--op-space-small);

    text-decoration: none;
    color: inherit;

    svg {
      width: var(--op-space-3x-large);
    }

    .sidebar__brand-label {
      display: flex;
      flex-direction: column;
      gap: var(--op-space-4x-small);
    }
  }

  .sidebar__content {
    gap: var(--op-space-x-small);
  }

  .sidebar__item {
    .sidebar__item-icon {
      border-radius: var(--op-radius-large);

      &.sidebar__item-icon--rotated {
        rotate: 90deg;
      }
    }

    &:hover,
    &.active {
      .sidebar__item-icon {
        --op-mso-weight: var(--op-font-weight-medium);
      }
    }
  }

  // Modifiers
  &.sidebar--drawer:not(.sidebar--responsive) {
    @include sidebar-16six-drawer-global;
  }

  &.sidebar--drawer.sidebar--responsive {
    @media (width > $breakpoint-medium) {
      @include sidebar-16six-drawer-global;
    }

    @media (width <=$breakpoint-medium) {
      @include sidebar-16six-rail-global;
    }
  }

  &.sidebar--rail {
    @include sidebar-16six-rail-global;
  }
}
```
