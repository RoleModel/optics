# Flash

<img src={imageFile} alt="Flash" />

Flash classes provide notification style alert messages that "Flash" onto the screen (top right corner) with application information.

**Note:** The `flash--demo` class that each example here uses is documentation only and not available in a real application. It is only used here to allow for showing the flash messages in a static way.

## Overriding flash styles

The flash classes are built on a sass placeholder selector https://sass-lang.com/documentation/style-rules/placeholder-selectors

This allows multiple flash classes to share the same behavior. You can modify all flash behavior by overriding the `%flash-global` placeholder selector and setting any properties:

```css
%flash-global {
  padding: var(--op-space-medium) var(--op-space-x-large);
}
```

## Custom Flash

Your application may need a custom flash. To add one, just follow this template:

```css
.flash-{name} {
  @extend %flash-global;

  background-color:
  color:
}
```

```css
.flash-purple {
  @extend %flash-global;

  background-color: purple;
  color: white;
}
```

You may need a new modifier of flash or other types of notifications. You can add one by overriding the `%flash-global` placeholder selector.

```css
%flash-global {
  &.flash--{name} {
    background-color:
    color:
  }
}
```
