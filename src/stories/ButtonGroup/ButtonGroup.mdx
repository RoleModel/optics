# Button Group

`.btn-group` is a class that can be added on a container around a collection of buttons. It provides grouping styles for any button conbination and variation that exists in the <LinkTo LinkTo kind="components-sidebar--default-drawer">Button Component</LinkTo>

## Button Group Toolbar

`.btn-group-toolbar` can be used around a collection of `.btn-group` containers to allow connected groups that wrap. A gap utility can be used to provide spacing between the groups.
