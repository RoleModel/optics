# Gap

Gap utility classes can be used to set the spacing between items within a flex or grid container.

## Cascade

When using a gap utility, `--rm-gap` will be set to the same spacing value as gap. This can be used in styles nested to match the container gap.

```css
.special-item {
  display: flex;
  gap: var(--rm-gap, var(--rm-space-sm)); // Include a fallback in case the wrapping gap is missing.
}
```

```html
<div class="flex gap-sm">
  <div>1</div>
  <div>2</div>
  <div class="special-item">
    <div>a</div>
    <div>b</div>
    <div>c</div>
  </div>
  <div>4</div>
</div>
```
