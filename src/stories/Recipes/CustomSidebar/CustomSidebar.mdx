## Domain Registrar Example

```css
// Domains Sidebar Example
%btn-global {
  &.btn--pill-right {
    border-radius: 0 var(--op-radius-pill) var(--op-radius-pill) 0;
  }
}

.sidebar-domains {
  @extend %sidebar-global;

  --_op-sidebar-drawer-width: 28rem;
  --_op-sidebar-content-spacing: 0;
  --_op-sidebar-content-item-spacing: 0;

  box-shadow: none;

  .btn {
    &.btn--no-border {
      box-shadow: none;

      &.btn--active {
        background-color: var(--op-color-primary-plus-five);
        color: var(--op-color-primary-on-plus-five);
      }

      &:hover:not(.btn--active) {
        background-color: var(--op-color-neutral-plus-six);
        box-shadow: none;
        color: var(--op-color-neutral-on-plus-six);
      }
    }
  }

  .sidebar__footer {
    display: flex;
    align-items: center;
    padding-left: var(--op-space-large);
    gap: var(--op-space-small);
  }
}
```

## Performance Management Software Example

```css
// 16Six Sidebar Example
:root {
  --op-color-primary-h: 256;
  --op-color-primary-s: 66%;
  --op-color-primary-l: 15%;

  // Custom
  --op-color-sidebar: var(--op-color-primary-minus-six);
  --op-color-on-sidebar: hsl(26deg 100% 95%);
  --op-color-sidebar-button-active: hsl(256deg 23% 32%);
}

.icon--rotated-135 {
  rotate: 135deg;
}

.icon--rotated-90 {
  rotate: 90deg;
}

@mixin sidebar-rail-global {
  .sidebar__brand {
    justify-content: center;

    .sidebar__brand-label {
      display: none;
    }
  }
}

.sidebar-16six {
  @extend %sidebar-global;

  --_op-sidebar-background-color: var(--op-color-sidebar);
  --_op-sidebar-text-color: var(--op-color-on-sidebar);
  --_op-sidebar-border-color: var(--op-color-on-sidebar);
  --_op-sidebar-rail-width: 6.4rem;
  --_op-sidebar-drawer-width: 22.4rem;
  --_op-sidebar-drawer-brand-width: calc(var(--op-space-3x-large) + (2 * var(--op-space-medium)));
  --_op-sidebar-brand-spacing: var(--op-space-medium) var(--op-space-x-small);
  --_op-sidebar-content-item-spacing: var(--op-space-x-small);
  --_op-sidebar-spacing: 0 0 var(--op-space-x-small);

  .sidebar__brand {
    display: flex;
    margin: 0;
    color: inherit;
    gap: var(--op-space-medium);
    text-decoration: none;

    svg {
      width: var(--op-space-3x-large);
      height: var(--op-space-3x-large);
    }

    .sidebar__brand-label {
      display: flex;
      flex-direction: column;
      font-size: var(--op-font-small);
    }
  }

  .btn {
    &.btn--no-border {
      background-color: transparent;
      box-shadow: none;
      color: var(--_op-sidebar-text-color);

      &.btn--active,
      &:hover {
        background-color: var(--op-color-sidebar-button-active);
      }
    }
  }

  &.sidebar--rail {
    @include sidebar-rail-global;
  }
}
```
