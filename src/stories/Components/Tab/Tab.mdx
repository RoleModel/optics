# Tab

Tab classes provide simple styling for a tab group navigation.

### Selective Imports

Tab can be used as a standalone component, however, it does have a few dependencies. To see a full dependency list, see [Dependency Graph](?path=/docs/overview-selective-imports--docs#dependencies)

```css
// Depends on
@import '@rolemodel/optics/dist/scss/core/tokens';
@import '@rolemodel/optics/dist/scss/core/base';

// Component
@import '@rolemodel/optics/dist/scss/components/tab';
```

## Variations

### Default

`.tab-group` Provides the grouping around a collection of links or buttons that have the `.tab` class.

Each item should use `.tab` and also supports `.tab--active` and `.tab--disabled`.

<Canvas of={TabStories.Default} />

### Small

A tab supports `.tab--small` or `.tab--large` with large being the default.

<Canvas of={TabStories.Small} />

## Tab API

Styles are built on css variables scoped to the tab.

Here are the variables that can be customized.

{/* prettier-ignore-start */}
```css
--_op-tab-font-small
--_op-tab-font-large
--_op-tab-padding-small
--_op-tab-padding-large
--_op-tab-indicator-width-small
--_op-tab-indicator-width-large
--_op-tab-disabled-opacity
```
{/* prettier-ignore-end */}

## Customizing Tab styles

<div

The tab classes are built on a [sass placeholder selector](https://sass-lang.com/documentation/style-rules/placeholder-selectors)

This allows multiple classes to share the same behavior. You can modify all tab behavior by overriding the `%tab-global` placeholder selector and setting any properties:

```css
%tab-global {
  --_op-tab-font-small: var(--op-font-2x-small);
  --_op-tab-padding-small: var(--op-space-3x-small) var(--op-space-small);
}
```

## New Tab Variations

  dangerouslySetInnerHTML={{

Your application may need a custom tab. To add one, just follow this template:

```css
.tab-{name} {
  @extend %tab-global;

  background-color: var(--op-color-primary-plus-two);
  color: var(--op-color-primary-on-plus-two);
}
```
