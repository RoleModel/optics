# Toggle

Toggle classes can be used to create a stylized checkbox or boolean input.

## Playground

<Canvas of={ToggleStories.Default} />
<Controls of={ToggleStories.Default} />

### Selective Imports

Toggle can be used as a standalone component, however, it does have a few dependencies. To see a full dependency list, see [Dependency Graph](?path=/docs/overview-selective-imports--docs#dependencies)

```css
// Depends on
@import '@rolemodel/optics/dist/scss/core/tokens';
@import '@rolemodel/optics/dist/scss/core/base';

// Component
@import '@rolemodel/optics/dist/scss/components/toggle';
```

## Variations

### Default

`.toggle` Provides basic styling for a checkbox or boolean input. It is intended to be used on an input element if type checkbox.

<Canvas of={ToggleStories.Default} />

### Size

`.toggle--large`, `.toggle--small` (with large being the default) modify the size of the toggle.

<Canvas of={ToggleStories.Small} />

### Disabled

`disabled='disabled'` will disable the toggle.

<Canvas of={ToggleStories.Disabled} />

## Toggle API

The size styles are built on css variables scoped to the toggle.

Here are the variables that can be customized.

{/* prettier-ignore-start */}
```css
--_op-toggle-height-small:
--_op-toggle-height-large:
--_op-toggle-width-small:
--_op-toggle-width-large:
--_op-toggle-opacity-disabled:
--_op-toggle-switch-padding:
```
{/* prettier-ignore-end */}

## Customizing Toggle styles

<div
  dangerouslySetInnerHTML={{

    }).outerHTML,

The toggle classes are built on a [sass placeholder selector](https://sass-lang.com/documentation/style-rules/placeholder-selectors)

This allows multiple classes to share the same behavior. You can modify all toggle behavior by overriding the `%toggle-global` placeholder selector and setting any properties:

```css
%toggle-global {
}
```

If you need to override the behavior of a particular toggle style, you can open the respective class and set or change properties

```css
.toggle-alternate {
}
```

Your application may need a custom toggle. To add one, just follow this template:

```css
.toggle-{name} {
  @extend %toggle-global;
}
```
