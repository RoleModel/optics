# Transition

Transition tokens can be used for transition or keyframe animation durations.

## Usage

These tokens can be applied as a transition duration.

```css
transition: all var(--op-transition-input);
// or
animation-duration: var(--op-transition-input);
```

## Available tokens and their definitions

<!-- prettier-ignore-start -->
```css
--op-transition-input:      120ms;
--op-transition-navigation: 200ms;
--op-transition-sidebar:    300ms;
--op-transition-modal:      300ms;
--op-transition-panel:      400ms;
--op-transition-flash:      5s;
```
<!-- prettier-ignore-end -->
