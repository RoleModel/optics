# Input Height

Input Height tokens are used to define a button or form controls height.

## Usage

These tokens can be applied to anything that requires a users input and needs a height.

```css
height: var(--op-input-height-large);
/* or */
max-height: var(--op-input-height-small);
/* or */
min-height: var(--op-input-height-medium);
```
